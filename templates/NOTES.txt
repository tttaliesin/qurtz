Qurtz has been installed successfully!

Your Quartz static site generator with NFS Obsidian vault monitoring is now running.

To access your site:
  kubectl --namespace {{ .Release.Namespace }} port-forward service/{{ include "qurtz.fullname" . }}-web 8080:80

Then visit: http://127.0.0.1:8080

To check the status:
  kubectl get pods --namespace {{ .Release.Namespace }}

To view logs:
  # File watcher logs
  kubectl logs --namespace {{ .Release.Namespace }} deployment/{{ include "qurtz.fullname" . }}-watcher -c file-watcher

  # Quartz builder logs  
  kubectl logs --namespace {{ .Release.Namespace }} deployment/{{ include "qurtz.fullname" . }}-watcher -c quartz-builder

  # Web server logs
  kubectl logs --namespace {{ .Release.Namespace }} deployment/{{ include "qurtz.fullname" . }}-web

Make sure to configure your NFS settings in values.yaml before installation!
